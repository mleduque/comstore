I. INSTALLATION

  1. Double-click the setup-Comstore.exe file and install to your main IWD2 directory
  2. Once the install program has finished, it will launce the WeiDU installer program. Simply follow the on-screen instructions in the new DOS window.

Though this mod is made using the WeiDU standard, and *should* be compatible with all other WeiDU-based mods, there is always the possibility for conflicts. We suggest you install this mod AFTER the official patches. If you encounter any problems installing the mod, please our forums.

II. UNINSTALLATION

Uninstalling is simple. Double-click the setup-Comstore.exe file and enter the given commands for Uninstallation.
Afterwards, you may delete the following files:

  • setup-Comstore.exe
  • setup-Comstore.tp2
  • setup-Comstore.debug
  • and the contents of the "comstore" subfolder

IV. FAQS

Q: What do I need to use this MOD?
A: The Chosen of Mystra Icewind Dale II Store Mod requires Icewind Dale II with the latest Interplay patches.

Q: Where can I find the new store clerk?
A: You can find Shi'nayne in the Weeping Widow Inn in Targos and she will appear in Kuldahar.

Q: What does this mod add?
A: This mod adds items from Baldur's Gate 2, Icewind Dale and some custom made ones that I made.
